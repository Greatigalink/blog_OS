# 努力向前的博客

## 运行

```bash
# 拉取代码
git clone https://github.com/Greatigalink/blog_OS.git
# 或者
git clone git@github.com:Greatigalink/blog_OS.git

# 安装依赖
npm install

# 开发环境（因为是开发，所以刷新速度会慢点，第一次打开需要等待）
npm run dev

# 生产环境（编译打包完后，打开超级块，秒开）
npm run build
npm start
```
